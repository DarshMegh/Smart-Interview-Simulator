# Smart Interview Simulator 🚀

A Java-based console application that simulates real interview scenarios and evaluates user performance using Data Structures.

---

## 🔥 Features
- Interactive question-based interview system  
- Topic-wise performance tracking  
- Real-time answer evaluation  
- Simulates real interview experience  

---

## 🧠 DSA Concepts Used
- Queue (LinkedList) → for managing question flow  
- HashMap → for tracking performance by topic  

---

## 🛠️ Tech Stack
- Java  
- Data Structures & Algorithms  
- IntelliJ IDEA  
- Git & GitHub  

---

## ▶️ How to Run
1. Open project in IntelliJ IDEA  
2. Run Main.java  
3. Answer questions in the console  

---
## 📸 Sample Output
```
Total Questions: 5
Question: What is JVM? Your Answer: java virtual machine
✅ Correct!
```
